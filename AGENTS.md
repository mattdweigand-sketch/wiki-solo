# AGENTS.md

This wiki's instructions live in [`CLAUDE.md`](CLAUDE.md). Read it first — same conventions apply regardless of which agent loaded this file.

The filename is duplicated so OpenAI Codex and other agents that auto-load `AGENTS.md` pick up the same map Claude Code reads from `CLAUDE.md`. The content is not maintained here; it forwards.

**Slash commands** (`/ingest`, `/lint`) are Claude Code's discovery convention. If your agent doesn't have them, follow the prose workflows in [`.claude/workspaces/ingest/CONTEXT.md`](.claude/workspaces/ingest/CONTEXT.md) and [`.claude/workspaces/maintenance/CONTEXT.md`](.claude/workspaces/maintenance/CONTEXT.md) instead — they describe the same steps without needing the shortcut.
